```bash
.           # repeat last command
c			# change
ciw         # change entire word (cursor in the middle)
C           # change to the end of the line (or use c$ command)
cw 		    # change + insertion (or use ce)
S           # change entire line (or use cc)
r           # replace a single char
s           # replace a single char and keep insert mode
R           # replace char until ESC (like insert key)
gU			# change to Upper
gu			# change to lower
g~			# switch case
u           # undo
ctrl + r    # redo
U           # restore last changed line
J           # join line bellow
gJ          # join line without space
xp          # transpose (swap two char)
```
